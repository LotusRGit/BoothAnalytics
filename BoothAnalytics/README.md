# Booth Analytics

Booth の注文 CSV から売上を分析するデスクトップアプリです。

---

## 機能

- **全体レポート** — 総売上・注文数・平均単価の KPI カード、売上推移グラフ、商品別ランキング
- **月次レポート** — 月ごとの切り替え表示、1位商品の大サムネイル、日別棒グラフ
- **時間帯分析** — 曜日×時間帯のヒートマップ、日付別購入傾向グラフ
- **リピーター分析** — ユニーク顧客数、リピーター率、購入回数分布
- **発売日表示** — 各商品の初回売上日・NEW バッジ・グラフへの縦線マーカー
- **商品サムネイル** — Booth の商品ページから自動取得

## 必要な環境

| ツール | バージョン |
|--------|-----------|
| Node.js | 18 以上 |
| npm | 9 以上 |

Node.js は https://nodejs.org/ja/ からインストールしてください。

## セットアップ

```bash
# 依存パッケージをインストール（初回のみ）
npm install
```

---

## 開発

### ブラウザで確認（推奨）

```bash
npm run dev
```

ブラウザで `http://localhost:5173` が開きます。

### Electron アプリとして起動

```bash
npm run electron:dev
```

Vite の開発サーバーと Electron を同時に起動します。

---

## ビルド（配布用）

```bash
npm run electron:build
```

`release\win-unpacked\` フォルダに実行ファイルが生成されます。

### 配布方法

`release\win-unpacked\` フォルダを **丸ごと zip 圧縮**して渡してください。
受け取った側は解凍後、フォルダ内の `Booth Analytics.exe` をダブルクリックするだけで起動できます。

> **注意**: `Booth Analytics.exe` 単体をフォルダの外に出すと起動しません。フォルダごと移動・配布してください。

### 起動中のアプリがある場合

ビルド前に `Booth Analytics.exe` を必ず終了してください。起動中はファイルがロックされてビルドが失敗します。

## 対応 CSV 形式

Booth 管理画面の **「注文管理」→「CSVダウンロード」** で出力したファイルに対応しています。

自動検出される列：

| 列名 | 用途 |
|------|------|
| 注文番号 | 重複排除 |
| 注文日時 | 日付・グラフの軸 |
| 商品名 | 商品グループ |
| 商品ID | サムネイル取得 |
| 単価 / 数量 / 小計 | 売上金額 |
| バリエーション名 | バリアント情報 |
| ユーザー識別コード | リピーター分析 |

複数月の CSV を一括読み込む場合は「フォルダを選択」ボタンを使うと自動で結合されます。同じ注文番号の重複は自動排除されます。

## サムネイル取得について

商品 ID をもとに Booth の商品ページから画像を取得します。
`npm run dev` または Electron アプリ起動中のみ動作します（内蔵プロキシを使用）。
取得できない場合は 📦 プレースホルダーが表示されます。
